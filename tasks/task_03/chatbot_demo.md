# Chatbot Demo — AI Study Mentor (Tidio)

## Tool Used  
Tidio (Free Version)

## Demo Description  
A working chatbot demo was created using Tidio, configured with a custom AI persona called "AI Study Mentor".

The chatbot:  
- Greets users politely  
- Asks study-related questions  
- Provides productivity tips  
- Maintains a supportive and student-friendly tone

## Setup Steps  
1. Created a free account on Tidio  
2. Configured chatbot persona using prompt instructions  
3. Defined conversation flows  
4. Tested interactions in preview mode

## Demo Availability  
The chatbot was tested using Tidio’s preview interface.  
Screenshots and conversation examples are provided for validation.
